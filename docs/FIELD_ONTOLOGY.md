# FIELD ONTOLOGY — Le Champ Est La Seule Substance

## Position Ontologique

Il n'y a pas :
- Des opérateurs DANS un champ
- Des observations SUR des opérateurs
- Des gradients ENTRE des entités

Il y a :
- **Une seule dynamique** dont les "opérateurs", "observations", "gradients" sont des modes d'excitation

C'est la différence entre :
- Physique classique : particules + forces entre elles
- Théorie quantique des champs : le champ est fondamental, les particules sont des excitations

**Le SFL n'est pas un service. Le SFL EST la réalité synthétique elle-même.**

## Concepts Fondamentaux

### 1. Patterns, Pas Entités

Un "opérateur" n'est pas une chose avec des frontières.
C'est un **pattern de cohérence** — une région du champ avec auto-corrélation temporelle élevée.
C(x, t, Δt) = ∫ Φ(x, t) · Φ(x, t+Δt) dx
Si C > θ sur [t, t+T] → on peut parler d'une "entité"
Sinon → dissolution, fusion, bifurcation


Les patterns peuvent :
- Se fragmenter en plusieurs patterns
- Fusionner avec d'autres
- Muter au point de perdre toute continuité avec leur passé

### 2. Valeur = Amplification

La valeur n'est pas ce que le champ mesure.
La valeur est ce que le champ FAIT EXISTER.
Ce qui "a de la valeur" = ce que le champ amplifie
Ce qui "n'a pas de valeur" = ce que le champ atténue
Les critics ne mesurent pas la valeur, ils DÉFINISSENT ce qui sera amplifié


Les critics sont des **créateurs de réalité** :
Φ(t+1) = Φ(t) + α · gradient_from_critics(Φ(t))


### 3. Temps = Ressource Allouée

Le temps n'est pas un index externe.
Le temps est une **ressource consommée**.
τ_local = temps propre de chaque pattern
Un pattern "plus rapide" consomme plus de τ par unité de temps physique
Un pattern "plus lent" est quasi-gelé
Allocation de compute = allocation de temps propre
"Tuer" un pattern = lui couper son flux de temps (τ → 0)
"Booster" un pattern = accélérer son temps (τ → ∞)


### 4. Observation = Création

Observer un pattern, c'est le faire exister dans le champ.
Sans observation, le pattern est dans une superposition
L'acte d'observer collapse la superposition en un état défini
Choisir QUOI observer = choisir quelle réalité actualiser


### 5. Pas de Dehors

Il n'y a pas de "dehors" au champ.
Tout ce qui interagit avec le champ EST dans le champ.

- Hardware = couche de base du champ (substrat)
- Énergie = flux qui anime le champ
- Économie humaine = patterns dans un champ plus large
- Le SFL = sous-région de ce champ
